# Система управління замовленнями ресторану

## Описання проєкту

Це об'єктно-орієнтований прототип системи управління замовленнями для ресторану, розроблений на Java. Проєкт демонструє основні принципи ООП:

- **Інкапсуляція**: приватні атрибути та публічні методи
- **Спадкування**: клас `Customer` наслідує `Person`
- **Поліморфізм**: інтерфейс `Ratable` та реалізація через `Order`
- **Абстракція**: абстрактний клас `Person` та реальні реалізації

## Компіляція та запуск

### Компіляція

Перейдіть до каталогу `src` та скомпілюйте всі Java-файли:

```bash
cd src
javac -d . *.java
```

Після компіляції буде створено каталог з скомпільованими класами.

### Запуск програми

Для запуску програми з каталогу `src`:

```bash
java -cp . Application
```

### Швидкий запуск (з кореневого каталогу проєкту)

```bash
# Компіляція та запуск в одній команді
cd src && javac -d . *.java && java -cp . Application
```

## Функціональність

Програма демонструє наступні операції:

1. **Реєстрація клієнтів** - створення нових клієнтів
2. **Управління меню** - додавання позицій меню
3. **Створення замовлень** - клієнти розміщують замовлення
4. **Управління позиціями** - додавання/видалення страв з замовлення
5. **Обчислення суми** - автоматичне розраховування вартості
6. **Управління статусом** - зміна статусу замовлення
7. **Оцінювання** - клієнти оцінюють замовлення (1-5 зірок)
8. **Статистика** - виведення загальної статистики ресторану

## Приклад використання

При запуску програма автоматично демонструє:

- Створення ресторану "Їжак" з меню
- Реєстрацію двох клієнтів
- Розміщення та обробку замовлень
- Зміну статусу замовлень
- Оцінювання замовлень клієнтами
- Виведення статистики ресторану

## Особливості реалізації

- ✅ **Мінімальна складність** - простий та зрозумілий код
- ✅ **Консольний вивід** - всі операції виводяться у консоль
- ✅ **Без зовнішніх залежностей** - використовуються лише стандартні бібліотеки Java
- ✅ **Без бази даних** - всі дані зберігаються в пам'яті
- ✅ **Українські коментарі** - весь код має українські коментарі для кращого розуміння
- ✅ **Англійські назви** - названия класів, методів та атрибутів англійською
